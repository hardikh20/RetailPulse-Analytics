RetailPulse Analytics
E-Commerce Intelligence Dashboard

---

Project Overview

RetailPulse Analytics is an end-to-end data analytics project focused on extracting meaningful business insights from e-commerce transaction data.
It demonstrates the complete data workflow — from raw data cleaning to advanced visualization.

---

Business Objectives

✔ Analyze monthly revenue trends
✔ Identify top-performing products
✔ Understand customer distribution
✔ Enable data-driven decision making

---

Tech Stack

Tool        | Purpose          
📊 Excel   | Data Cleaning
🗄️ SQL     | Data Querying
🐍 Python  | EDA & Analysis
📈 Power BI| Dashboard & Visualization
