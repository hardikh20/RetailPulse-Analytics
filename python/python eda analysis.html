E-Commerce Retail Sales Analysis¶

This project analyzes an online retail dataset to understand sales performance, customer behavior, and revenue trends.
The analysis focuses on identifying top-performing products, key revenue-generating markets, purchasing patterns, and customer segments using data analytics techniques.

Tools Used:

  • Python
  • Pandas
  • NumPy
  • Matplotlib
  • Seaborn

Objectives:

  • Understand overall sales performance
  • Identify top products and countries by revenue
  • Analyze monthly sales trends
  • Segment customers using RFM analysis

r_labels = range(4,0,-1)
f_labels = range(1,5)
m_labels = range(1,5)
rfm["R"] = pd.qcut(rfm["Recency"],4,labels=r_labels)
rfm["F"] = pd.qcut(rfm["Frequency"],4,labels=f_labels)
rfm["M"] = pd.qcut(rfm["Monetary"],4,labels=m_labels)

rfm.head()
rfm["RFM_Score"] = rfm["R"].astype(int) + rfm["F"].astype(int) + rfm["M"].astype(int)

In [70]:
def segment_customer(row):
    
    if row["R"] == 4 and row["F"] >= 3:
        return "Champions"
    
    elif row["R"] >= 3 and row["F"] >= 3:
        return "Loyal Customers"
    
    elif row["R"] >= 3 and row["F"] <= 2:
        return "Potential Loyalists"
    
    elif row["R"] == 2:
        return "At Risk"
    
    else:
        return "Lost Customers"

rfm["Segment"] = rfm.apply(segment_customer, axis=1)
